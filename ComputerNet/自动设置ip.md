# 手动设置 IP 地址操作指南


## 一、Windows 系统设置方法

### 方法：通过图形界面设置

1. 按 `Win + R`，输入 `ncpa.cpl`，回车打开 **网络连接**。
2. 右键当前使用的网络（如“以太网”）→ 选择 **属性**。
3. 在列表中双击 **Internet 协议版本 4 (TCP/IPv4)**。
4. 选择 **“使用下面的 IP 地址”**，填写：
   - IP 地址：`192.168.18.172`
   - 子网掩码：`255.255.255.0`
   - 默认网关：`192.168.18.1`
5. 在下方填写 DNS 服务器：
   - 首选 DNS 服务器：`192.168.18.10`
   - 备用 DNS 服务器：`114.114.114.114`
6. 点击 **确定** → **关闭**。

✅ 设置完成后，打开命令提示符执行 `ipconfig` 验证配置。

### 恢复自动获取（DHCP）
- 在相同界面选择 **“自动获得 IP 地址”** 和 **“自动获得 DNS 服务器地址”**。

# 网络四要素：IP、子网掩码、网关、DNS（精简版）

在有线/无线网络中，设备要正常上网或通信，需正确配置以下四个参数：

---

## 1. IP 地址
- **作用**：设备在网络中的唯一标识（如 `192.168.18.172`）。
- **类比**：门牌号。
- **类型**：
  - 内网 IP（私有）：`192.168.x.x`、`10.x.x.x`、`172.16~31.x.x`
  - 公网 IP：全球唯一，由运营商分配。

> ✅ 同一局域网内不能重复。

---

## 2. 子网掩码
- **作用**：划分“内网范围”，判断目标是否在同一局域网。
- **常见值**：`255.255.255.0`（即 `/24`）
- **示例**：
  - IP：`192.168.18.172`
  - 掩码：`255.255.255.0`
  - → 同网段范围：`192.168.18.1` ~ `192.168.18.254`

> ✅ 访问同网段设备直连；访问外网需经网关。

---

## 3. 默认网关
- **作用**：访问外网（如互联网）的出口，通常是路由器 IP（如 `192.168.18.1`）。
- **关键点**：
  - 无网关 → 能连内网，但**无法上外网**。
  - 必须与本机 IP 在同一网段。

> 🚪 类比：小区大门，出外必经之路。

---

## 4. DNS 服务器
- **作用**：将域名（如 `www.baidu.com`）解析为 IP 地址。
- **常见 DNS**：
  - 公共：`114.114.114.114`、`8.8.8.8`
  - 企业：`192.168.x.10`（用于解析内部系统）
- **问题表现**：
  - 能 ping 通 IP，但打不开网页 → **DNS 故障**

> 📞 类比：114 查号台。

---

## 协同工作流程
1. 输入 `www.qq.com`  
2. 问 DNS 要 IP  
3. 判断目标 IP 是否在本地网段（用子网掩码）  
4. 若不在 → 数据发给**默认网关**  
5. 网关转发到互联网，返回结果

---

## 配置建议
| 场景 | 建议 |
|------|------|
| 家庭用户 | 使用 **自动获取（DHCP）** |
| 公司/固定设备 | 按 IT 要求**手动设置**，尤其注意 DNS |

> ⚠️ 非必要勿手动改 IP，避免冲突或断网。
