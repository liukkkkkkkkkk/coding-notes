# python读取元素
## 正向走就是0开始，负向走就是-1开始
### 截取的时候是左开右闭规则
#### 例子：
```python
# 字符串（str）
s = "hello"
print(s[0])   # 'h'      → 第一个字符
print(s[-1])  # 'o'      → 最后一个字符
print(s[1:4]) # 'ell'    → 切片：从索引1到3
```
```python
#列表
lst = [10, 20, 30, 40]
print(lst[0])    # 10
print(lst[-1])   # 40
print(lst[1:3])  # [20, 30]
# 还可以嵌套访问

matrix = [[1, 2], [3, 4]]
print(matrix[0][1])  # 2
```
```python
#字典 通过键取值
d = {'name': 'Alice', 'age': 25}
print(d['name'])        # 'Alice'
print(d.get('age'))     # 25（推荐用 .get() 避免 KeyError）
print(d.get('height', 0))  # 0（若 key 不存在，返回默认值）
```
```python
#集合（set),不能直接"读取单个元素"
集合是无序且不重复的，不支持索引，所以不能像 s[0] 这样读取。
但可以遍历或弹出一个元素：
s = {1, 2, 3}
for x in s:
    print(x)  # 顺序不确定

# 或取出一个任意元素
elem = s.pop()  # 移除并返回一个元素（会改变集合！）

```
```python
#反转字符串

s = "hello"
print(s[::-1])  # 输出: 'olleh'

```
