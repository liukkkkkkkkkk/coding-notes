## find() 是 Python 字符串（str）对象的一个内置方法，用于查找子字符串在主字符串中第一次出现的位置（索引）。(列表跟元组不能使)
### 找到了就返回最小的索引（从0开始）| 没找到就返回-1
### 不会抛出异常，这是它和.index()最大的区别
### 基础语法：
```python
str.find(sub[, start[, end]])
```
