# C# `try-catch` 异常处理详解

> 异常是程序运行时发生的错误事件。`try-catch` 是 C# 中用于捕获和处理这些异常的核心机制，能有效防止程序崩溃并提升健壮性。

## 什么是 `try-catch`？

`try-catch` 是 C# 提供的结构化异常处理语法，用于：
- 捕获运行时错误（如空引用、除零、文件未找到等）
- 避免程序因未处理异常而意外终止
- 执行自定义错误恢复或日志记录逻辑

---

## 基本语法

```csharp
try
{
    // 可能抛出异常的代码
}
catch (SpecificException ex)
{
    // 处理特定类型的异常
}
catch (Exception ex)
{
    // 兜底：处理所有其他未被捕获的异常
}
finally
{
    // 无论是否发生异常都会执行（可选）
}
```
## 示例：
```c#
try
{
    int number = int.Parse(Console.ReadLine()); // 可能抛出 FormatException
    int result = 10 / number;                   // 可能抛出 DivideByZeroException
    Console.WriteLine($"结果: {result}");
}
catch (FormatException ex)
{
    Console.WriteLine("输入内容不是有效数字！");
}
catch (DivideByZeroException ex)
{
    Console.WriteLine("不能除以零！");
}
catch (Exception ex)
{
    Console.WriteLine($"未知错误: {ex.Message}");
}
finally
{
    Console.WriteLine("操作已完成。");
}
```
