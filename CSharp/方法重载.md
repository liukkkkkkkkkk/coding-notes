# 方法重载
## 定义: 在同一个类中，定义多个同名方法，但它们的参数列表不同（参数个数、类型或顺序不同）。编译器会根据调用时传入的参数，自动选择最匹配的那个方法。

### 代码示例:

```c#
 // 版本1：指定行号
private void setValue(string fieldName, object value, int row)
{
    Model.SetValue(fieldName, value, row);
    View.UpdateView(fieldName);
    View.InvokeFieldUpdateService(fieldName, row);
}

// 版本2：不指定行号，默认第0行（重载！）
private void setValue(string fieldName, object value)
{
    setValue(fieldName, value, 0); // 调用版本1
}
调用时:
setValue("FPrice", 100m);        // 自动调用版本2 → 再转到版本1（row=0）
setValue("FPrice", 100m, 2);     // 直接调用版本1（row=2）

```
